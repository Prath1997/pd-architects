# PD Architects Website

## Overview
This project is the official website for **PD Architects** (Pradeep Deshmukh Architects), Nashik. It showcases the firm's philosophy, services, and projects while providing a user-friendly experience. The website is designed to be SEO-friendly and includes smooth scrolling and animations for enhanced interactivity.

## Project Structure
```
architect-website
├── images
├── css
│   └── style.css
├── js
│   └── script.js
├── index.html
├── about.html
├── services.html
├── contact.html
└── README.md
```

## Features
- **Responsive Design**: The website is fully responsive and adapts to various screen sizes.
- **Sticky Navigation Bar**: Easy navigation throughout the site with a sticky header.
- **Hero Section**: Engaging hero section with a background image.
- **About Section**: Detailed information about the firm's philosophy and team.
- **Services Section**: Grid layout showcasing various services offered.
- **Project Gallery**: A section dedicated to displaying the firm's projects.
- **Contact Form**: A user-friendly contact form for inquiries.
- **Map Integration**: Displays the firm's location on the contact page.
- **Smooth Scrolling**: Enhances user experience with smooth transitions between sections.
- **Animations**: Basic animations for interactive elements.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd architect-website
   ```
3. Open `index.html` in your web browser to view the website.
